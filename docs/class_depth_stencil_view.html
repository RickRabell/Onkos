Detailed Description

Manages a Direct3D 11 Depth Stencil View (DSV).

Author
Ricardo Rabell
Date
2025-09-25

This class encapsulates an ID3D11DepthStencilView. The depth-stencil buffer is used for depth testing (to ensure correct object occlusion) and stencil testing (for more advanced effects like masking or outlining).
