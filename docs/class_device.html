Detailed Description

A wrapper class for the DirectX 11 ID3D11Device interface.

Author
Ricardo Rabell
Date
2025-09-11

This class encapsulates the ID3D11Device object, which is responsible for creating resources such as textures, buffers, and shaders. It acts as a factory for all GPU resources within the engine.
