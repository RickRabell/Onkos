Detailed Description

A wrapper class for the DirectX 11 ID3D11DeviceContext interface.

Author
Ricardo Rabell
Date
2025-09-12

This class encapsulates the ID3D11DeviceContext object, which is responsible for generating rendering commands. It sets pipeline state and issues drawing calls to the GPU.
