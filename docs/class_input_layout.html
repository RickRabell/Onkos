Detailed Description

Manages a Direct3D 11 Input Layout object.

Author
Ricardo Rabell
Date
2025-10-09

This class encapsulates an ID3D11InputLayout. The input layout's critical role is to describe how vertex data from a vertex buffer is streamed into the Input-Assembler (IA) stage of the graphics pipeline. It defines the format, size, and semantic meaning of each element in a vertex (e.g., position, color, texture coordinate) so that it correctly matches the input signature of the vertex shader.
