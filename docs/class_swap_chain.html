Detailed Description

Manages the front and back buffers for rendering and presentation.

Author
Ricardo Rabell
Date
2025-09-19

This class encapsulates the IDXGISwapChain, which is a fundamental component of DirectX graphics. It holds a collection of buffers (typically a front buffer displayed on the screen and a back buffer for rendering) and is responsible for presenting the rendered image to the user.
