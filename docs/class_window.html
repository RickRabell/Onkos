Detailed Description

Manages the creation and lifecycle of the application's main window.

Author
Ricardo Rabell
Date
2025-09-05

This class encapsulates the functionality related to a Win32 window, including its initialization, message loop handling (via update), and destruction. It holds the window handle (HWND) and its dimensions.
