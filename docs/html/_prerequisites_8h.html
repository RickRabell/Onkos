Detailed Description

A precompiled header file for the engine.

Author
Ricardo Rabell
Date
2025-09-05

This file includes commonly used headers from the Standard Library, Windows API, and DirectX 11. It also defines several utility macros and data structures that are used throughout the project. It is intended to be included in nearly all source files to reduce compilation times and simplify dependency management.
