Detailed Description

A data component holding CPU-side geometry.

Author
Ricardo Rabell
Date
2025-10-17

This class represents the raw geometry for a 3D model, storing the list of vertices and indices in system memory (on the CPU). It acts as a data container that is typically used by a Buffer to create GPU-side vertex and index buffers. It is likely part of an Entity-Component System (ECS) architecture.
