Detailed Description

Manages a Direct3D 11 Render Target View (RTV).

Author
Ricardo Rabell
Date
2025-09-25

This class encapsulates an ID3D11RenderTargetView, which represents a resource (typically a texture) that the GPU can render to. It acts as a drawing canvas for the graphics pipeline.
