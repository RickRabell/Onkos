Detailed Description

Manages a Direct3D 11 Sampler State object.

Author
Ricardo Rabell
Date
2025-10-29

This class encapsulates an ID3D11SamplerState. The sampler state defines how texture data is sampled by a shader. It controls parameters like filtering (e.g., linear, point, anisotropic), addressing modes (e.g., wrap, clamp, mirror), and mipmap LOD (Level of Detail) biasing.
