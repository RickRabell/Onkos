Detailed Description

Manages a Direct3D 11 Viewport configuration.

Author
Ricardo Rabell
Date
2025-10-09

This class encapsulates a D3D11_VIEWPORT structure. The viewport defines the 2D rectangle area on a render target to which a 3D scene is projected. It's used by the rasterizer stage to map vertices from normalized device coordinates to window coordinates.
