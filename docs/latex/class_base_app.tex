\doxysubsection{Detailed Description}
The core application class that initializes and runs the engine. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/29

\doxylink{class_base_app}{Base\+App} is responsible for owning and managing the lifecycle of all primary graphics objects (\doxylink{class_device}{Device}, \doxylink{class_device_context}{Device\+Context}, \doxylink{class_swap_chain}{Swap\+Chain}, etc.), handling the main message loop, and orchestrating the per-\/frame update and render calls. It serves as the central hub for the entire application.
