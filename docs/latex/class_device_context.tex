\doxysubsection{Detailed Description}
A wrapper class for the DirectX 11 ID3\+D11\+Device\+Context interface. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/09-\/12
\end{DoxyDate}
This class encapsulates the ID3\+D11\+Device\+Context object, which is responsible for generating rendering commands. It sets pipeline state and issues drawing calls to the GPU.
