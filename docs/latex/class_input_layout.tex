\doxysubsection{Detailed Description}
Manages a Direct3D 11 Input Layout object. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/09
\end{DoxyDate}
This class encapsulates an ID3\+D11\+Input\+Layout. The input layout\textquotesingle{}s critical role is to describe how vertex data from a vertex buffer is streamed into the Input-\/\+Assembler (IA) stage of the graphics pipeline. It defines the format, size, and semantic meaning of each element in a vertex (e.\+g., position, color, texture coordinate) so that it correctly matches the input signature of the vertex shader.
