\doxysubsection{Detailed Description}
Manages a Direct3D 11 Render Target View (RTV). 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/09-\/25
\end{DoxyDate}
This class encapsulates an ID3\+D11\+Render\+Target\+View, which represents a resource (typically a texture) that the GPU can render to. It acts as a drawing canvas for the graphics pipeline.
