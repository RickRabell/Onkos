\doxysubsection{Detailed Description}
Manages a Direct3D 11 Sampler State object. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/29
\end{DoxyDate}
This class encapsulates an ID3\+D11\+Sampler\+State. The sampler state defines how texture data is sampled by a shader. It controls parameters like filtering (e.\+g., linear, point, anisotropic), addressing modes (e.\+g., wrap, clamp, mirror), and mipmap LOD (Level of Detail) biasing.
