\doxysubsection{Detailed Description}
A wrapper for DirectX 2D texture resources (ID3\+D11\+Texture2D). 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/09-\/18
\end{DoxyDate}
This class handles the creation, management, and binding of 2D textures. It supports loading textures from image files and creating procedural textures for use as render targets or depth-\/stencil buffers.
