\doxysubsection{Detailed Description}
Manages a Direct3D 11 \doxylink{class_viewport}{Viewport} configuration. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/09
\end{DoxyDate}
This class encapsulates a D3\+D11\+\_\+\+VIEWPORT structure. The viewport defines the 2D rectangle area on a render target to which a 3D scene is projected. It\textquotesingle{}s used by the rasterizer stage to map vertices from normalized device coordinates to window coordinates.
