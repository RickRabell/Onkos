\doxysubsection{Detailed Description}
Manages the creation and lifecycle of the application\textquotesingle{}s main window. 

\begin{DoxyAuthor}{Author}
Ricardo Rabell 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/09-\/05
\end{DoxyDate}
This class encapsulates the functionality related to a Win32 window, including its initialization, message loop handling (via update), and destruction. It holds the window handle (HWND) and its dimensions.
